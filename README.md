## Run Instructions (Web & Mobile)

Follow these steps to run the project locally for development and to build artifacts for production.

### Prerequisites

- Install Flutter: https://docs.flutter.dev/get-started/install
- Ensure `flutter doctor` reports no blocking issues.

```bash
flutter doctor
```

### Run on Web (Chrome)

```bash
flutter pub get
flutter run -d chrome
```

Build production web bundle:

```bash
flutter build web --release
```

### Run on Android

```bash
flutter pub get
flutter run -d <device-id>
```

Build release APK:

```bash
flutter build apk --release
```

### Run on iOS (macOS only)

```bash
flutter pub get
flutter run -d <ios-device-id>
```

Open Xcode workspace for signing/debugging:

```bash
open ios/Runner.xcworkspace
```

### Helpful Commands

```bash
flutter pub get
flutter clean
flutter build web --release
flutter build apk --release
```
